# Intro
Een vuurpijl wordt van een weegschaal afgeschoten. 

# Wat geeft de weegschaalaan aan?
- Een negatief getal.
- Een positief getal
- Het getal verandert niet
    
# Verklaring
Het getal geeft de stuwkracht van de vuurpijl af. Dit is een positief getal.
